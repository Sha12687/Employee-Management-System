# Employee Management System
This project is a simple Employee Management System implemented using Spring Boot and JSP (JavaServer Pages). It allows you to perform basic CRUD (Create, Read, Update, Delete) operations on employee records.

## Features
Add a new employee with details like name, address, email, phone number, and salary.
View a list of all employees with their respective details.
Edit employee details, including name, address, email, phone number, and salary.
Delete an employee from the system.

## Technologies Used
Java
Spring Boot
JSP (JavaServer Pages)
MySQL (or any other supported database)
HTML/CSS
Bootstrap
JavaScript
